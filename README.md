🤖 Gemini LLM App - Q&A Chatbot

📌 Overview

This project is a Q&A chatbot application built using Streamlit and powered by Google's Gemini-Pro model. It allows users to ask questions and receive AI-generated responses, leveraging Google Generative AI.

🔍 Key Features

🚀 LLM Integration:

Uses Google's Gemini-Pro model for generating responses.

Loads API keys securely using dotenv.

🎨 Interactive UI:

Developed using Streamlit for a user-friendly interface.

Provides real-time responses to user queries.

🔧 Custom Functionality:

Accepts user input via a text box.

Processes queries and returns AI-generated answers.

Simple and efficient app design for rapid deployment.
